# React Product Card

## Description
Projet React simple qui affiche une carte de produit avec React Bootstrap.  
Le projet contient des composants séparés pour afficher le nom, le prix, la description et l'image du produit.

Un message est aussi affiché :
- "Hello + nom" si le nom existe
- "Hello, there!" sinon  
Et une image de profil s'affiche si le nom est fourni.

---

## Components
- Name
- Price
- Description
- Image

---

## Technologies
- React
- React Bootstrap
- CSS

---

## Run project
npm install  
npm start
